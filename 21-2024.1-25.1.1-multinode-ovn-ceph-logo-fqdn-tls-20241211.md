## Ceph

- Ceph Version : 18.2.4

### List pool

```bash
ceph osd pool ls
# .mgr
# ceph

ceph osd lspools
# 1 .mgr
# 2 ceph
```

### Create pool

```bash
ceph osd pool create volumes
ceph osd pool create images
ceph osd pool create backups
ceph osd pool create vms
```

### Set Application

```bash
ceph osd pool application enable volumes rbd
ceph osd pool application enable images rbd
ceph osd pool application enable backups rbd
ceph osd pool application enable vms rbd
```

```bash
ceph osd pool application get volumes
ceph osd pool application get images
ceph osd pool application get backups
ceph osd pool application get vms
```

### Delete pool

```bash
ceph osd pool rm volumes volumes --yes-i-really-really-mean-it
ceph osd pool rm images images --yes-i-really-really-mean-it
ceph osd pool rm backups backups --yes-i-really-really-mean-it
ceph osd pool rm vms vms --yes-i-really-really-mean-it
```

### Init pool

> 초기화 하지 않아도 사용 가능. 참고용.

```bash
rbd pool init volumes
rbd pool init images
rbd pool init backups
rbd pool init vms
```
